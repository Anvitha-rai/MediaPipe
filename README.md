MediaPipe Computer Vision Detection System

This repository contains a computer vision detection system built using MediaPipe, OpenCV, and NumPy. The project includes multiple modules for detecting faces, hands, body pose, and iris landmarks in real-time using a webcam.

All modules are controlled through a central main.py file, which manages and runs the different detection scripts.

Features

Face Detection
Detects faces in real time using MediaPipe face detection.

Hand Detection
Tracks hand landmarks and detects hand movements.

Pose Detection
Detects full body pose and key body joints.

Iris Detection
Detects eye landmarks and iris position for eye tracking.

Central Controller (main.py)
The main.py file controls and runs all detection modules.
